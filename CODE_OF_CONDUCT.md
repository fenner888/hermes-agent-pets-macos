# Code of Conduct

Be respectful, direct, and constructive.

This project welcomes people building local-first Hermes companion tools. Keep discussion focused on improving the software, documentation, installation flow, and user safety.

Unacceptable behavior includes harassment, personal attacks, doxxing, deliberate disruption, or submitting assets you do not have the right to share.

Project maintainers may remove issues, comments, or contributions that violate this code of conduct.
